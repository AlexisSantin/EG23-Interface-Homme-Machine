# Prompt – Team Template

> Ce fichier contient le ou les prompts IA utilisés pour générer l'interface utilisateur.
> Remplacez l'exemple ci-dessous par vos propres prompts.

---

## Prompt initial (< 60 secondes)

```
Crée une application web de gestion de tâches moderne et épurée avec un thème sombre.
Elle doit avoir un en-tête avec le nom de l'application, un champ de saisie pour ajouter des tâches,
une liste de tâches avec des cases à cocher et des boutons de suppression, et un compteur de tâches.
Utilise uniquement HTML, CSS et JavaScript. Rends-la responsive.
```

---

## Itération 1

```
Ajoute des animations fluides lors de l'ajout ou de la suppression de tâches.
Utilise une couleur d'accent violette (#6c63ff) pour les éléments interactifs.
Ajoute un bouton "Effacer les tâches complétées" en bas.
```

---

## Itération 2

```
Ajoute la persistance via le localStorage pour que les tâches survivent à un rechargement de page.
Affiche un message d'état vide motivant quand il n'y a aucune tâche.
```
