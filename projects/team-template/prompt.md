## Initial Prompt (< 60 seconds)

```
Create a clean, modern task manager web app with a dark theme.
It should have a header with the app name, an input field to add tasks,
a list of tasks with checkboxes and delete buttons, and a task counter.
Use only HTML, CSS, and JavaScript. Make it responsive.
```

---

## Iteration 1

```
Add smooth animations when tasks are added or removed.
Use a purple accent color (#6c63ff) for interactive elements.
Add a "Clear completed" button at the bottom.
```

---

## Iteration 2

```
Add local storage persistence so tasks survive a page refresh.
Display a motivational empty-state message when there are no tasks.
```
